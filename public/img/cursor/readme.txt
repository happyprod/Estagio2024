=== LLX Purple XL Cursor Set ===

Author's description:

The "LLX Purple XL" cursor set is an extra large set of 15 cursors. All the cursors are purple. Purple cursors are amazing and hard to find. This cursor set has a very thin pink border which is occasionally visible when staring close to the edge of the cursors.
